# project_advanced_sql
Projeto: Simulação de banco de dados de uma instituição financeira fictícia, com foco em análise de clientes, transações, contas e empréstimos utilizando SQL avançado.  O objetivo é demonstrar domínio de consultas complexas, análises de negócio e técnicas avançadas de SQL, aplicadas a um contexto realista do setor financeiro.
